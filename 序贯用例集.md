# 序贯用例集（P6-1·测试用例集中含真实复诊/服药反馈记录的案例）

筛选标准：案例本身包含"初诊处方→服药后反馈→胡老据反馈调整"的完整记录（非单纯"继服"），
共选7案，覆盖：单纯剂量调整/加味调整/换方调整三种复杂度。均为回顾模式(已发表医案)。

## 序案1｜蔡某·男48岁（案015）—— 换方级调整
- **初诊**(1964-11-23)：半月高烧腹痛，中药10余剂不效。自汗盗汗甚，腹痛剧甚，胃脘亦痛，
  午后高烧40℃，苔白微腻，脉沉弦紧。证属里饮郁久化热。
  **胡老初方**：附子粳米汤合小半夏加茯苓汤(温阳化饮)。
- **服后反馈**(3剂后·11-26二诊)：腹痛减，胃痛、高烧如故，仍汗出多，且恶风明显，脉数而虚。
- **胡老N诊调整**：里寒虽稍减，而表虚不固→改黄芪建中汤(温中固表)。服3剂烧渐退汗出已减，
  继服3剂热平身凉和，**但晚上仍腹痛肠鸣**→再与11-23方(附子粳米汤合小半夏加茯苓汤)调之。
  12-5告之：腹痛已。
- **金标准终态**：附子粳米汤合小半夏加茯苓汤 ⇄ 黄芪建中汤 交替(表里各有侧重，随症轮用)。

## 序案2｜刘某·女23岁（案041）—— 加合方级调整
- **初诊**(1965-3-15)：五年浮肿时常低烧，面目四肢皆肿，小便频量少色黄，大便时干，
  低烧时恶寒、腹胀、右胁痛、头晕心烦，脉浮微数。证属水气外郁肌肤(溢饮)。
  **胡老初方**：大青龙汤加苍术。
- **服后反馈**(30余剂后)：头晕心烦减，面目浮肿减，**午后仍低热，下肢浮肿仍明显**。
- **胡老N诊调整**：继加减服用，或间服柴胡桂枝干姜汤合当归芍药散(转治少阳枢机+血虚水盛)。
  1965-11-7复诊：右胁痛减，腹胀、头晕、心烦已，下肢浮肿轻微，尿常规基本转阴。
- **金标准终态**：大青龙汤加苍术(初) → 间服柴胡桂枝干姜汤合当归芍药散(续治残留低热+浮肿)。

## 序案3｜任某·女33岁（案017，与案009同患者·病历号131）—— 剂量+加味调整
- **初诊**(1966-3-25)：八九年腰背疼痛(脊椎骨质增生)，头昏头痛目胀，下肢关节疼手麻木，
  全身无力，四肢逆冷，苔白润，脉沉细。证属寒湿痹痛，表虚湿重。
  **胡老初方**：桂枝加术附汤。
- **服后反馈**(6剂后)：腰痛稍减，**他症无变化**。
- **胡老N诊调整**：上方加茯苓12克继服。一周后痛麻皆减，继服原方。
  4-15复诊：痛麻已不明显，天气变化时也不加重。
- **金标准终态**：桂枝加术附汤 → +茯苓(苓术附结构补全，同G4/A1丁某案·刘某案模式)。

## 序案4｜张某·女65岁（案103）—— 加味调整(养血基础上补充)
- **初诊**(1965-12-13)：多年失眠久治无效，头晕口干心悸心烦汗出，重时连续不得眠，
  苔白，舌质红少津，脉虚数。证属阴血虚损，阳不得入于阴。
  **胡老初方**：酸枣仁汤加生牡蛎生龙骨。
- **服后反馈**(3剂后)：睡眠已稍安，**但心悸烦、自汗出、头晕口干不欲饮等仍明显**。
- **胡老N诊调整**：上方加当归10克、白芍12克、桂枝10克、白术10克(转向兼顾营卫/血虚)，
  继服3剂，一切症状均消，为巩固疗效继服3剂。
- **金标准终态**：酸枣仁汤加龙牡 → +当归白芍桂枝白术(养血敛阴基础上加调营卫)。

## 序案5｜李某·女36岁（案099）—— 单味剂量滴定
- **初诊**(1966-5-6)：产后左偏头痛三年未愈，时心下痛，左上下肢酸胀，口干不思饮，
  有时恶心吐清水，苔白润，脉弦细。证属表虚饮虚。
  **胡老初方**：当归四逆加吴萸生姜汤(吴萸10克)。
- **服后反馈**(4剂后)：头痛明显减轻，心下痛未作，左上下肢酸胀亦减(**方向正确，未达痊愈**)。
- **胡老N诊调整**：上方增吴萸为12克，继服7剂，已自感无不适。
- **金标准终态**：同方，仅吴茱萸剂量上调(10→12克)，无需换方——纯剂量滴定型序贯。

## 序案6｜刘某·男36岁（案072）—— 单味加味(不换方向)
- **初诊**(1966-5-6)：左小腿阳部痛疼腰亦强急不适，经中西药治疗一年多不效，口中和不思饮，
  苔白润，脉弦迟。证属寒饮困滞、经筋失养。
  **胡老初方**：大黄附子汤合芍药甘草汤。
- **服后反馈**(6剂后)：腰强急减，**遇劳则剧痛**(仍有残留)。
- **胡老N诊调整**：上方加苍术12克，服6剂，腰强急基本愈，继服一月诸症不复作。
- **金标准终态**：大黄附子汤合芍药甘草汤 → +苍术(祛湿力度加强，方向不变)。

## 序案7｜刘某·男44岁（案014）—— 剂量+加味调整
- **初诊**(1964-9-9)：胃穿孔术后大便溏泄未已，肠鸣腹痛腰痛两足拘急，头晕乏力心悸短气，
  汗出如流，曾多次昏迷(贫血)，舌无苔，脉沉细。证属表里俱虚、卫弱血衰。
  **胡老初方**：当归建中汤加苍术泽泻。
- **服后反馈**(3剂后)：诸证减，**唯心悸气短尚明显**。
- **胡老N诊调整**：增桂枝为12克，加生龙牡各15克，继服，诸症渐渐好转。
  11-30复诊：除脘腹胀外，余无所苦。
- **金标准终态**：当归建中汤加苍术泽泻 → 桂枝加量+加龙牡(针对残留心悸气短，心阳浮越/血虚)。

## 覆盖类型统计
- 换方级(病位/病性方向转变)：序案1、序案2（2例）
- 加合方级(主方外加其他方向)：序案3(苓术附补全)、序案4(养血加调营卫)、序案7(加龙牡)（3例）
- 单味加减/剂量滴定(不改变主方向)：序案5、序案6（2例）
