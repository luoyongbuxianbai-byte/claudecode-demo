# 待裁决清单

本会话在已核验的约140条候选中，未遇到"纲领无法明判"的真正[待裁决]红线案例——
凡遇取舍分歧，均可用现有纲领(五·温病立场 / 六·体系裁决 / A4.5属性完整 / A4.6主诉优先)
直接裁定，故本清单当前为空。

保留本清单文件是因为：
1. 队列中仍有约1600余条未经人工核验(见 挂载日志.md 末尾"本会话实际完成范围"一节)，
   后续批次处理时如遇纲领无法明判的案例，应记录于此，不阻塞队列处理(规则R3)。
2. 已知一个需要人工确认的方向性问题，先记录在案：
   - 【经方理论(全量)】章节含大量【辨证要点】【方解】【仲景对本方证的论述】等结构化
     小标题内的B/C级候选，与【验案】区块高度交织。本会话T1重抽时未按这些小标题
     二次归类候选来源类型(辨证要点/方解/医案按语)，只按"章节"粗粒度归类。
     后续批次处理经方理论(全量)剩余候选时，建议先用这些小标题进一步定位，
     避免将【方解】中的机理性文字与【辨证要点】中的判据性文字混为一谈误判弃留。
     此点不影响已挂载的5条(均逐条核实过小标题上下文)，仅提示后续处理注意力分配。
